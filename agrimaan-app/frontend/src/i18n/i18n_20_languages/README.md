This package contains your original `hi.json` and a script to automatically translate it into 20 languages.

## Target language codes
- `bn`: Bengali
- `mr`: Marathi
- `te`: Telugu
- `ta`: Tamil
- `gu`: Gujarati
- `ur`: Urdu
- `kn`: Kannada
- `or`: Odia
- `ml`: Malayalam
- `pa`: Punjabi
- `as`: Assamese
- `mai`: Maithili
- `sat`: Santali
- `ks`: Kashmiri
- `ne`: Nepali
- `sd`: Sindhi
- `doi`: Dogri
- `kok`: Konkani
- `mni`: Meitei (Manipuri)
- `sa`: Sanskrit

## Quick start
```bash
cd i18n_20_languages
npm i
node scripts/translate.mjs
```
